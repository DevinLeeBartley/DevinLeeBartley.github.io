MAP672: Programming for Web Mapping

Web Cartography by Devin Bartley

Map of Zero Emission Power Plants in the United States

This is a Leaflet.js based web map with interactive features created in JavaScript. The map was created by converting the power generation capcity values to proportional symbols. In JavaScript I created a spotlight filter with a 500 km radius that appears when the user clicks on the map. The filter also summarizes information about power generation capacities visible within the spotlight.

  • See the Map: Zero Emission Power Plants in the United States
  • Data Source: US Energy Information Administration
  • GitHub Repo: GitHub - Zero Emission Power Plants


Map of Breweries in North America

This map is a hexbin heat map created in QGIS by using a spatial join method. The map demonstrates that hexbins create a visually appealing heat map highlighting brewing culture hotspots around the country.

  • See the Map: Breweries of North America
  • Data Source: POI Factory





Map of Cancer Rates by County in the United States

This map was created in CartoDB by joining the cancer profiles tables to US Census Bureau Tiger linework. The map showcases custom labels and color schemes created using CSS and HTML within the CartoDB framework. The map interactivity allows the user to investigate cancer rates for all counties in the country, and the counties with the highest and lowest rates are clearly labelled. Using SQL, I reprojected the map to the Albers Equal Area projection to escape the ubiquitous Web Mercator projection.

  • See the Map: Cancer Rates by County in the United States
  • Data Source: State Cancer Profiles




Map of The Pacific Crest Trail - Displayed on Landforms of the United States by Erwin Raisz

This map was created by georeferencing the iconic Landforms of the United States by Erwin Raisz in QGIS, and then overlaying Pacific Crest Trail Linework and Natural Earth Administrative boundaries on top. I used Didot typface for the map and highlighted the states that the trail passes through in a shade of green that matches the trail logo.

  • See the Map: Pacific Crest Trail on Landforms of the United States
  • Data Source: US Forest Service
