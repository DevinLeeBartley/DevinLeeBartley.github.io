Zero Emissions Power Plants in the United States

This map is an example of a leaflet based web map with interactive features created in Javascript

This map was created by changing the power generation capcity numbers in the power plant dataset to proportional symbols. In Javascript I created a spotlight filter with a 500 km radius is created when the user clicks on the map, and summarizing statistics about the area within the filter are displayed. Also unique popups were created to show the name and generation capacity of individual powerplants.

  • See the Map: Zero Emissions Power Plants in the United States
  • Data Source: US Energy Information Administration




Map of Breweries in North America

This map is an example of using hexbins to display point density by area.

This map was created in QGIS using a spatial join method. Since the hexbin areas are of equal size, they produce a visually appropriate heat map showing the density of points by area.

  • See the Map: Breweries of North America
  • Data Source: POI Factory




Map of Cancer Rates by County in the United States

This map was created in CartoDB by joining the cancer profiles tables to US Census Bureau Tiger county linework. This map showcases some custom labels and color schemes are created using CSS and HTML. I reprojected the map to Albers Equal Area by using an SQL query within the CartoDB layer.
